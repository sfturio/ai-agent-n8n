1. Stack Tecnológica

Backend

  • Node.js
  • Express
  • JavaScript (ESModules)
  • Arquitetura em camadas (routes, controllers, services)

Integração

  • n8n (engine de automação via webhook)

Frontend (Interface de Teste)

  • HTML
  • CSS
  • JavaScript puro (DOM)

Deploy

  • Render
  • Git & GitHub

2. Arquitetura

Frontend → POST /agent → Controller → Service → Webhook n8n → JSON Response

3. Conceitos Aplicados

  • Separação de responsabilidades
  • Roteamento modular
  • Controle de fluxo com async/await
  • Validação de inputs
  • Códigos HTTP (200, 400, 500)

4. Funcionalidades Implementadas

  • Endpoint REST (POST /agent)
  • Arquitetura backend em camadas
  • Comunicação assíncrona com webhook
  • Respostas em JSON
  • Interface web estilo chat para testes
  • Deploy em ambiente de produção

5. Principais Bugs e Correções

Erro 500 (Internal Server Error)

  • Lógica assíncrona não tratada corretamente
  • Retorno inconsistente no service
  • Validação de resposta do webhook

Problemas com Router

  • Padronização do uso do Express Router

Problemas de Deploy (Render)

  • Ajuste do start script no package.json
  • Configuração correta da versão do Node

Limpeza do Repositório Git

  • Remoção de remote duplicado
  • Reorganização de commits

6. Resultado

  • Backend modular funcional
  • Integração com workflow externo (n8n)
  • Deploy ativo
  • Experiência real com debugging em produção
