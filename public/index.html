About Project

PRO

AI Agent Portfolio Project

General AI agent para demonstração de arquitetura web, integração com automação e observabilidade operacional.

Visão Geral

O AI Agent é um projeto de portfólio com interface de chat e backend orientado a integração. Ele utiliza um agente de IA geral via fluxo no n8n, sem fine-tuning ou treinamento proprietário deste projeto. A solução combina Node.js + Express, frontend leve em HTML/CSS/JS, persistência em Supabase Postgres e painel com métricas reais de execução.

Arquitetura
  • Backend em Node.js (Express) para roteamento de requisições e controle de fallback.
  • Integração assíncrona com n8n via webhook para geração de resposta do agente.
  • Persistência de mensagens e metadados em Supabase Postgres.
  • Deploy em Render com variáveis de ambiente separadas por serviço.
Diferenciais Técnicos
  • Fallback interno para manter o chat funcional quando o fluxo externo falha.
  • Retry com backoff para 429 e rate limit por IP para maior estabilidade operacional.
  • Arquitetura desacoplada por webhook, facilitando troca de provedor de IA no futuro.
  • Persistência estruturada das interações para análise de uso e rastreabilidade.
  • Métricas reais no dashboard: total de execuções, erros e tempo médio de resposta.
Roadmap
  • Expandir dashboard com histórico temporal e filtros por período.
  • Adicionar novas integrações de entrada (logs, tickets e eventos operacionais).
  • Melhorar trilha de auditoria e qualidade de resposta com métricas comparativas.

Status atual: versão v1.3.0 em operação estável como projeto de portfólio, com foco em confiabilidade, observabilidade e evolução incremental.
